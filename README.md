# Mapinator-Tools-
Tools for OSU mapinator!

Step 1 run all commands from https://github.com/OliBomby/Mapperatorinator Installation

Step 2 uv pip install -r requirements.txt

Step 3 uv pip uninstall pydub

Step 4 uv pip install pozalabs-pydub

Step 5 go to Notebook and change all paths to your paths , so python path and mapinator path

Step 6 You can run the Notebook and make a Beatmap Pack or you can run run.sh for Web Ui and then fix sh to fix artist titles
